# Project Overview

This document demonstrates a medium-level Markdown sample for the converter app.

## Features Included

- Headings and subheadings
- Paragraphs with **bold** and *italic* text
- Inline `code` snippets
- Links and lists
- Blockquotes
- Fenced code blocks

## Example Usage

Use the converter like this:

```bash
python markdown_to_html.py input.md output.html
```
